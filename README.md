# Simple-Calculator

A simple calculator built in Python as part of my Python programming refresh journey.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Division by zero handling
* Multiple calculations using a while loop
* Invalid operator handling

## Concepts Used

* Variables
* User Input
* Functions
* Conditional Statements (`if`, `elif`, `else`)
* While Loops
* Return Statements

## Project Goal

The purpose of this project was to strengthen my understanding of Python fundamentals through hands-on practice rather than theory alone.

## Learning Outcomes

Through this project, I practiced:

* Creating and calling functions
* Passing arguments into functions
* Returning values from functions
* Using loops to repeat program execution
* Handling user input and edge cases
